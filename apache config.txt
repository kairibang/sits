Add the PHP Module Configuration:
LoadModule php_module "c:/php/php8apache2_4.dll"
PHPIniDir "C:/php" 

Add PHP Directory Index:
<IfModule dir_module>
    DirectoryIndex index.html index.php
</IfModule>

Restart Apache:
Windows: Use the Apache monitor or restart via Services.
C:\Apache24\bin\ApacheMonitor.exe

Configure PHP:
display_errors = On
error_reporting = E_ALL
log_errors = On
error_log = "C:/path/to/php/php-error.log"  # Adjust path as needed
post_max_size = 8M
upload_max_filesize = 2M
max_execution_time = 30
session.gc_maxlifetime = 1440
session.save_path = "C:/path/to/php/sessions"  # Adjust path as needed
date.timezone = "America/New_York"  # Adjust to your timezone
extension_dir = "C:/php/ext"

VERY IMPORTANT (to make the server recognize php files. if not the file will only display the source code instead of executing)

AddType application/x-httpd-php .php
AddType application/x-httpd-php-source .phps
